Select 選擇器

2022-05-16, Yii

# 標題 內容

3 Ant Design https://ant.design/components/select-cn/
4 Web #
5 App #

定義

用於收取大量選項的組件。

何時使用

需要在有限的空間內提供多種展示選項，並在選擇時提供多個顯示選項。

位置

位於卡片內。

規則

  • 當選項少於 6 個，建議直接將選項平鋪，請改用 Radio 單選。
  • 右側的 X 可清除已選擇的條件。
一般選擇器
  • 單選項目建議設定至多10個。
  • 單選超過請改用 Search +Select 搜尋單選元件。
  • 複選項目超過 6個 以上，且需選多個項目時建議選用此選單。
  • 複選條件採聯集(OR)方式查詢。如：包含A條件或B條件的相關資料。
搜尋選擇器
  • 單選 or 複選
  • 用於可選擇項目較多、資料庫已有資料，但需查詢時使用此組元件。
  • 可於下拉選單框內以關鍵字搜尋選項。
  • 輸入字串後顯示關聯字，除下拉選擇外，也可只輸入關鍵字模糊搜尋。
    如商品編號輸入1，在input中點一下、或離開input，都會增加"1"的搜尋字串

類型

1.分組

在不同層次的選擇器，能夠對不同層次的用戶提供方便的選擇關係。

2.地址
