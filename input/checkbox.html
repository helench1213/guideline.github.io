Checkbox 多選框

2022-05-13, Yii

# 標題 內容

3 Ant Design https://ant.design/components/checkbox-cn/

5 App #

定義

多選框是一個選擇元件。

何時使用

需要從一個數據中，選擇多個選項時。

位置

卡片內

規則

  1. Check Box 預設 全部不選。
  2. 若只有一個項目表示時，可只使用一個Check Box。如：無限制時間
  3. 建議至多設置 6個 Check Box，超過6個請建議使用複選元件
  4. 當Check Box勾取多個時，採聯集(OR)方式查詢符合條件的所有資料，
    顯示於列表中。如：A、B、C三個核取框，當勾選A、B時，表示查詢包含A或B相關的所有資料。
  5. 可於最前端設置 [全部] 選取的Check Box，當全部被勾選時同條件的項目一併被選取。

類型

列表：第一列選取
表單：至多設置6個
表單：可單獨使用
