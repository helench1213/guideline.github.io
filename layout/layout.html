1 Status 框架

定義

頁面級別的整體框架。

規則

頁面的基本框架頁面由以下四項元素所阻成。

  • Content：主要內容區域，由卡片集合而成。
  • Header ：詳細規範請見 Breadcrumb 麵包屑 。
  • Sider：詳細規範請見 Menu 選單。
  • Footer：版權宣告僅出現於列表頁底部。

類型

以下僅粗略列出系統中最常見的版型，詳細規範請至各頁查看。

1. 列表頁

詳細規範請見 列表頁 。


2. 編輯、新增頁

詳細規範請見 新增編輯。


3. 對話框（彈窗）

詳細規範請見 Modal 對話框。


4. 抽屜

詳細規範請見 Drawer 抽屜 。
