定義

UI 構成的元素，作為一定程度上影響了 UI 中重要的外觀呈現風格。

何時使用

需要圖樣輔助文字，或是代替文字時，使用 Icon 來視覺化文字，以減少使用者閱讀的負擔。

位置

根據出現位置與功能不同，有不同的 Icon，可至下方「樣式」區塊查看。

規則

檔案類型皆為向量 svg 檔，可至 zeplin 下載。

大部分 Icon 皆有功能且可點選，僅有少部分作為示意（例：警告、錯誤、提示）使用。
